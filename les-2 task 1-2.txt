


1. Дан код:
var a = 1, b = 1, c, d;
Почему код даёт именно такие результаты?

c = ++a; alert(c); // 2  значение увеличивается на 1, далее возвращается эта сумма (2) 
d = b++; alert(d); // 1  Значение возвращается, а увеличивается как бы за перделами видимости.Мы видим ответ 1.
c = (2+ ++a); alert(c); // 5 складываем 2 и значение a (а=2 после инкремента 2,т.е.2+(2+1)), получаем 5 
d = (2+ b++); alert(d); // 4 складываем 2 и значение b (2) итого 4, при этом b=2, но 1 в уме
alert(a); // 3 значение после последнего инкремента
alert(b); // 3 значение после последнего инкремента
